# Records-Recommendations

## Personal Project: a Vinyl Record Recommendation application

Record, manage, and recommend vinyl records based on users collection and
listening history. Stores collection and history in iCloud to allow for
syncing across different devices.

### Main Recommendation Page

![Recommendation-page](./Images/Recommendation.png)

Shows off daily and weekly recommendations as well as simple stats on the number
of times a record has been played.

### Vinyl Collection Edit-page

![Collection-page](./Images/Collection.png)

Showcase full collection along with play statistics.

### New Vinyl Page

![Edit-page](./Images/New.png)

Allows for edits to record info or play counts.
